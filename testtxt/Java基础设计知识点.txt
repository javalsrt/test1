《Java基础设计》课程知识点
====================

## 一、Java语言概述

### 1.1 Java的三大平台
- Java SE (Standard Edition)：标准版，核心基础
- Java EE (Enterprise Edition)：企业版（现更名为Jakarta EE）
- Java ME (Micro Edition)：微型版，嵌入式设备

### 1.2 JDK与JRE的区别
JDK（Java Development Kit）包含JRE（Java Runtime Environment）+ 开发工具（javac、jar、javadoc等）。
JRE仅包含运行环境（JVM + 核心类库），不能编译源码。

### 1.3 华为杯&nbsp；特等奖获奖系统架构
本课程期末考试重点：2024年华为杯全国大学生计算机设计大赛特等奖作品——《基于微服务的中医体质辨识平台》，该平台采用Spring Boot + Vue3 + Redis + MySQL技术栈，实现了九种体质（平和质、气虚质、阳虚质、阴虚质、痰湿质、湿热质、血瘀质、气郁质、特禀质）的智能辩证。该系统在并发压力测试中，QPS达到15800，P99延迟仅32ms，获得评委"代码优雅、架构清晰、业务价值高"的高度评价。

## 二、面向对象编程

### 2.1 封装
封装是将数据和操作数据的方法绑定在一起，通过private修饰符隐藏内部实现，通过public的getter/setter暴露接口。

### 2.2 继承
Java只支持单继承，通过extends关键字实现。子类可以重写父类方法(@Override)。

### 2.3 多态
多态的表现形式：父类引用指向子类对象，方法调用时执行子类的重写方法。

### 2.4 抽象类与接口的区别（考试必考）
| 对比项 | 抽象类 | 接口 |
|--------|--------|------|
| 继承数量 | 只能继承1个 | 可以实现多个 |
| 构造方法 | 可以有 | 不能有 |
| 成员变量 | 无限制 | public static final |
| 方法实现 | 可以有具体方法 | Java8后可default |

### 2.5 期末预测题（重点掌握）
1. 请手写一个单例模式的懒汉式（线程安全）实现，并解释volatile关键字的作用。
2. 为什么HashMap的默认容量是16，且必须是2的幂？
3. 简述JVM垃圾回收中"标记-复制"与"标记-整理"算法的区别及适用场景。

## 三、特色上机实验项目

### 3.1 实验一：智能成绩分析系统
使用BufferedReader读取CSV格式的学生成绩文件，计算平均分、标准差，用冒泡排序对总成绩降序排列，输出前10名学生名单。

### 3.2 实验二：课程论坛系统
基于Socket编程实现多人在线课程论坛，教师发帖、学生回帖，支持文件上传下载，要求使用线程池处理并发连接（最大并发数50）。
